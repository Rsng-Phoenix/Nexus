# NEXUS v3.4

A Priority-Driven Productivity App built around Matrix Listing.

Unlike traditional to-do apps that focus on endless lists, Nexus helps you focus on what matters most by organizing tasks into priority-based quadrants.

---

## Core Features

### Matrix-Based Task Management

- Create, edit, and delete tasks
- Organize tasks into:
  - High Priority
  - Medium Priority
  - Low Priority
  - No Priority
- Matrix-style task organization
- Drag and drop tasks between quadrants
- Reorder tasks within a quadrant
- Pin important tasks
- Mark tasks as completed
- Archive completed work through task history

---

## Notes System

Each task can contain rich notes.

Supports:

- Plain text
- Checklists
- Bullet lists
- Numbered lists
- Nested content
- Long-form planning notes

Perfect for:

- Study planning
- Project tracking
- Daily journals
- Meeting notes

---

## Advanced Reminder System

### Standard Reminders

- Exact date and time reminders
- Notification opens directly to the task

### All-Day Reminders

Receive reminders repeatedly throughout a day.

Examples:

- Every hour
- Every 2 hours
- Every 4 hours
- Custom interval

### Date Range Reminders

Create recurring reminders across multiple days.

Examples:

- Exam preparation period
- Assignment deadlines
- Workout programs
- Habit tracking

### Pinned Task Notifications

Pinned tasks can display persistent notifications until completed.

### Notification Actions

- Mark Done
- Snooze
- Open Task
